# Simulation Projects

This folder contains simulation exercises for statistical computing course.

## Files

- **Simulation_Exercise_10.Rmd**: Non-parametric statistical tests
  - Sign Test
  - Runs Test
  - Run length counting

- **Simulation_Chapter5_Exercise12.Rmd**: Distribution simulation
  - Weibull distribution using inverse transform method
  - Comparison of theoretical and empirical PDF/CDF
  - Custom discrete distribution simulation
  - Histogram method for continuous distributions

- **Direct_Transformation_Methods.Rmd**: Direct transformation methods
  - Inverse transform method for Cauchy and t-Student distributions
  - Box-Muller method for generating standard normal variables
  - Shapiro-Wilk normality test
