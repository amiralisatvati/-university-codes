# Probability and Stochastic Processes

This folder contains probability calculations and stochastic process simulations.

## Files

- **Stochastic_Process_Exercise.Rmd**: Markov chain simulation with R
  - Markov path simulation with transition matrix
  - Detection of absorbing/transient/recurrent states
  - Bernoulli process simulation (reaching 30 successes)
  - Random Walk simulation (until reaching boundary)

- **Probability_Exercise_With_R.Rmd**: Probability calculation exercises with R
