Stochastic Process Analysis - Markov Chains

This document contains analysis of Markov chains and stochastic processes.

1. Markov Chain Simulation

Simulating state transitions in a Markov process with transition matrix P.

# Transition matrix
P <- matrix(c(0.7, 0.3,
              0.4, 0.6), nrow=2, byrow=TRUE)

# Markov chain simulation
markov_sim <- function(n, P, init) {
  states <- numeric(n)
  states[1] <- init
  for(i in 2:n) {
    states[i] <- sample(1:ncol(P), 1, prob=P[states[i-1],])
  }
  return(states)
}

2. Stationary Distribution

Calculating the stationary distribution π where πP = π.

# Solve for stationary distribution
eig <- eigen(t(P))
