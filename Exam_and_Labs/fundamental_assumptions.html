Fundamental Assumptions of Inferential Methods

Key Assumptions

  1. Independence: Observations are independent of each other
  2. Normality: Data follows a normal distribution
  3. Homoscedasticity: Equal variances across groups
  4. Linearity: Linear relationship between variables

Diagnostics

# Normality test
shapiro.test(residuals)

# Homoscedasticity test
bptest(model)

# Independence test
durbinWatsonTest(model)
