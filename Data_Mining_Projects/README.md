# Data Mining Projects

This folder contains machine learning and data visualization projects.

## Files

- **Data_Mining_Exercise_2.Rmd**: Machine learning project for predicting faculty ranks
  - Preprocessing: NA removal, variable conversion to Factor
  - CART model with pruning based on optimal CP
  - Random Forest model with 200 trees
  - Evaluation: Confusion Matrix, 87.8% accuracy (CART), 90% accuracy (RF)
  - Variable importance analysis

- **Discrete_Data_Analysis.Rmd**: Graphical analysis with 5 chart types
  - Bar chart: Titanic passengers analysis
  - Polar Bar chart: Spotify music genre popularity
  - Treemap: Music subgenre comparison
  - Heatmap: Car features correlation (mtcars)
  - Radar/Spider chart: Pop song features comparison
